# Wave-9 사전등록 스펙 — $100 네이티브 · 단기 · 수익최대화 (동결: 2026-07-21)

## 왜 새 wave인가
wave1~8은 목적함수가 "안정성(샤프·MDD 최소화)"이고 구조가 $300 기준 다중레그/다종목이었다. wave-8 판정: **$100에서 기존 후보 전원 실행 불가**(캐리는 현물+퍼프 gross 2x, 모멘텀은 레그 $0.10 < 최소주문 $5). 이 wave는 처음부터 $100 제약 안에서 설계한다.

## 구조 제약 (하드)
- **단일레그 퍼프만** — 현물 헤지 금지(gross 2x 회피). Bitget USDT-M.
- 총자본 $100, 현금버퍼 10% → 활성자본 $90. **동시 포지션 ≤2** (포지션당 ≥$45 노셔널 → 최소주문 $5 여유 충족).
- 레버리지 ≤3x (wave-4 실측: 3x 초과 시 청산이 수익을 지배). 청산모델 wave-4 계약 재사용.
- 보유기간 ≤7일 (사용자 요구: 기간 짧게).

## 목적함수 (기존 wave와 다름 — 명시)
maximize **MC 중앙값 최종자본**, subject to:
- (H1) 파산확률 P(최종 < $30) < 20%
- (H2) MC 5%분위 > $50 (최악 5%에서 절반 이상 보존)
- (H3) OOS(2025-10~, 현 레짐) 비용후 수익 > 0
- (H4) 실행가능성: 전 포지션 ≥$5, gross ≤ 활성자본×레버리지, 단일레그
- (H5) 블록셔플(90일) MDD p95 ≤ 50% — 고변동 프로파일이므로 한도 완화하되 명시
**샤프·Calmar는 참고 지표로만 기록**(안정성이 목적이 아님). H1·H2·H4는 타협 불가.
