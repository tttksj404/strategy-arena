# Wave-2 사전등록 스펙 (동결: 2026-07-16)

Wave-1 판정 근거: 캐리 패밀리만 구조 생존(F1e +68.9%/22트레이드), 사인은 "비용"과 "가동률". Wave-2는 비용 반감 + 가동률 개선 6변형 + H3 재검증. **이 7개가 전부, 사후 추가 금지.**

## 공통 변경
- 메이커 체결 가정 변형: 수수료 0.02%/레그, 슬리피지 0 (지정가 진입·청산; 미체결 리스크는 진입 1바 지연 유지로 보수 근사). 표기 [M].
- 분할·비용·체결·게이트 나머지는 wave-1 SPEC 그대로 상속. OOS 2025-10-01~ 동결 유지.

## 판정
- W2a~W2f: wave-1과 동일 게이트 표. 단 "OOS 무포지션"은 FAIL이 아닌 **UNTESTED_IN_OOS**로 별도 표기하고, IS 연도별 일관성(게이트3)과 IS 비용후 +EV(게이트4의 IS 버전)를 병기.
- W2g: 효과 통계 + 비용·베이시스 후 수익 부호. 배포 청구 금지(탐색등급).
- 생존 정의: [M] 가정이 핵심이므로 생존 후보는 "메이커 체결률 실측" 없이는 배포 금지 → paper 단계에서 지정가 체결률 수집이 후속 게이트.
