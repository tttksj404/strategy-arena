# Wave-5 사전등록 스펙 — 캐리 휴면기 보완 전략 (동결: 2026-07-16)

목표: W2c(캐리)가 현금 대기하는 저펀딩 레짐에서 수익을 내는 **무상관 보완 전략**. 판정의 최종 기준은 단독 성과가 아니라 **W2c와의 결합 개선**이다.

## 제약
- Bitget 실행 가능(퍼프 위주 — 현물 공매도 불요 구조만), $300에서 W2c와 병행 가능한 마진 사용량.
- 데이터·비용·체결·분할 규약은 wave-1 상속(IS ~2025-09 / OOS 2025-10~). OOS가 곧 캐리 휴면 구간이라 목표 적합성을 직접 검정.
- **다중검정 규율**: 아래 7후보가 전부, 사후 추가 금지. W5f는 기존 OOS에서 이미 관찰된 W3c의 파생이므로 **OOS 오염 명시 — IS 일관성으로만 판정**하고 OOS 수치는 참고 표기.

## 판정
- 단독: 19게이트 표준 표 + UNTESTED_IN_OOS 라벨 규약.
- 결합(W5g, 최종 목표): corr(후보, W2c) < 0.3 ∧ 결합 MDD ≤ W2c 단독 ×1.5 ∧ 결합 CAGR > W2c 단독 ∧ OOS(휴면기) 결합 수익 > W2c 단독(≈0).
- 통과 없으면 "휴면기 보완 없음 — W2c 단독 + 현금 대기가 최적" 정직 보고.
